# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of the Election Guide AI very seriously. If you find a security vulnerability, please do not report it publicly. Instead, please report it via the GitHub Issues marked as "Security" or contact the maintainers directly.

We will acknowledge your report within 24 hours and provide a fix as soon as possible.
